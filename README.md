# Merge-Sort
Dizide Merge sort algoritması kullanımı: <br>
[16,21,11,8,12,22] dizisi için,<br>
    [16,21,11]               [8,12,22]<br>
                                   
   [16,21]  [11]            [8,12]   [22]<br>
                                     
  [16] [21]   [11]         [8] [12]    [22]<br>
                                 
   [16,21]    [11]           [8,12]    [22]<br>
                                 
   [11,16,21]              [8,12,22]<br>
                            
   [8,11,12,16,21,22]<br>
                  
O(nlogn)
